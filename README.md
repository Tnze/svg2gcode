# svg2gcode

Convert SVG image to G-code

## Support feature

- [ ] \<rect\>
- [ ] \<circle\>
- [ ] \<ellipse\>
- [ ] \<line\>
- [ ] \<polyline\>
- [ ] \<polygon\>
- [x] \<path\>
    - [x] M, m
    - [x] L, l
    - [x] C, c
    - [ ] T, t
    - [ ] Q, q
    - [ ] S, s
    - [ ] A, a
    - [ ] Z, z

> support more later
